## 📖 Overview

Welcome to my repository **FastAPI**, a structured collection documenting my step-by-step learning of FastAPI and its ecosystem, including essential technologies like Docker and related tools.  
A step-by-step journey into FastAPI, from building simple endpoints to developing full-fledged API-backed project (insurance_premium_category_predictor) with frontend interfaces and ML model integration. This repository is packed with hands-on projects, scripts, and notes documenting my learning journey along the way.


> “I started with basic endpoints and ended up exploring how FastAPI powers modern web APIs, data handling, and backend intelligence.”

> “Project Based Learning Experience”

The notes and scripts in this repository represent real learning — written, tested, and refined through consistent study and experimentation.

# 📚 Skills Acquired

## ⚡ FastAPI Framework
- Built high-performance REST APIs with automatic interactive documentation (Swagger UI)  
- Integarted request validation and data modeling using Pydantic (UserInput schema) for structured and type-safe inputs.
- Implemented path parameters, query parameters, and request body validation  
- Created asynchronous endpoints for improved performance   

## 📋 Pydantic Data Validation
- Defined data models with type hints and automatic validation  
- Created custom validators and field constraints for robust data handling  
- Implemented Field Transformation and Normalization:
- Designed request/response schemas for API endpoints  
- Integrated Computed Properties and Field Validators
- Performed serialization and deserialization of complex data structures using JSON 

## 🎨 Streamlit Frontend Development
- Built interactive web interfaces for real-time model integration
- Created user-friendly forms, and input widgets  
- Displayed real-time data visualizations and model predictions  
- Integrated Streamlit with FastAPI backend services  

## 🐳 Docker Containerization
- Built a Docker image using python:3.13.3 and copied project files to /app.
- Installed dependencies from requirements.txt and additional packages like scikit-learn.
- Exposed ports 8000 (FastAPI) and 8501 (Streamlit).
- Executed commands.sh to launch both backend and frontend.
- Ensures a consistent, reproducible environment for development and deployment. 
- Docker pull command ( run via terminal )
> docker pull manusharansah/insurance_premium_predictor:latest
## ☁️ AWS Deployment 
- Gained understanding of AWS services such as EC2, ECS, ECR, and Elastic Beanstalk.
- Learned cloud deployment workflows, container management, and CI/CD concepts.
- Prepared to deploy containerized applications to the cloud in future projects. 

## 🤖 ML Model Deployment
- Serialized trained ML models using Pickle for production use  
- Loaded and served models through FastAPI
- Handled model predictions with proper error handling  
- Optimized model inference for real-time responses
